## Description

Inventory Managment to manage a clinical laboratory

## Tecnologies

- TypeScript
- React
- Next.js
- Postgres
- Shadcn
- Prisma

## Requirements

- Node.js ^20.17
- Pnpm

## Installation

1. Install dependencies:

```bash
$ pnpm install
```

2. Run application:

```bash
# development watch mode
$ pnpm dev

# production mode
$ pnpm build
$ pnpm start
```
